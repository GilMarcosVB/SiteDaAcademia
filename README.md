# 🏋️ Site da Academia

> Projeto final do curso Técnico de Informática para Internet (Mobile).  
> Desenvolvido 100% pelo aluno, do zero ao deploy local.

---

## 📋 Sobre o projeto

Site completo para uma academia, desenvolvido como projeto de conclusão do curso técnico. O projeto começou em **PHP puro** e evoluiu para uma arquitetura mais robusta com **Laravel** e **Bootstrap**, simulando um ambiente de desenvolvimento profissional real.

Além do desenvolvimento, o projeto envolveu a configuração completa do ambiente local com **Laragon** — incluindo o primeiro contato com **variáveis de ambiente** e configuração de servidor local.

---

## 🚀 Evolução do projeto

| Fase | Tecnologia | Descrição |
|---|---|---|
| 1ª fase | PHP puro | Estrutura inicial, lógica do servidor e páginas dinâmicas |
| 2ª fase | Laravel + Bootstrap | Refatoração com framework MVC e interface responsiva |

## ⚙️ Ambiente de desenvolvimento

O projeto foi executado localmente com **Laragon**, onde aprendi a:

- Configurar um servidor local para rodar PHP e Laravel
- Trabalhar com **variáveis de ambiente** (arquivo `.env`)
- Configurar banco de dados local
- Simular um ambiente de desenvolvimento profissional na própria máquina

---

## 💡 Aprendizados

- Desenvolvimento web com **PHP puro** — entendendo a base antes do framework
- Migração para arquitetura **MVC com Laravel**
- Design responsivo com **Bootstrap**
- Configuração de ambiente local com **Laragon**
- Primeiro contato com **variáveis de ambiente** — conceito fundamental no desenvolvimento profissional
- Organização de projeto do início ao fim como projeto real
